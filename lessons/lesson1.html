Програма / Урок 2

Перший світлодіод.
Система «Світлофор»

⏱ 45 хв·📚 5–7 класи·Статус: у процесі
Природничі науки і технології Математична Інформаційно-цифрова Уміння вчитися

🎯 Мета уроку

Після цього уроку ти зможеш:

  • пояснити, як працюють макетна плата, світлодіод і резистор;
  • правильно зібрати схему й відрізнити анод від катода;
  • написати програму, що керує блиманням світлодіода;
  • використати цикл for, щоб повторювати дії;
  • зібрати власну систему «Світлофор».

🧩 Що знадобиться

Обери будь-який шлях — обидва ведуть до однакового результату.

🧰 Реальний набір

На столі

  • Плата Arduino UNO + USB-кабель
  • Макетна плата та дроти
  • 3 світлодіоди (червоний, жовтий, зелений)
  • 3 резистори 220 Ом

⚠️ Техніка безпеки

Збирай і змінюй схему лише тоді, коли плата від'єднана від комп'ютера. Не з'єднуй «+» і «−» напряму. Світлодіод завжди підключай через резистор.

📖 Теорія коротко

Макетна плата

Це плата для швидкого складання схем без паяння. Отвори з'єднані між собою за прихованими правилами: уздовж країв ідуть довгі лінії живлення (червона «+» і синя «−»), а в центрі отвори з'єднані короткими рядами поперек плати. Саме тому компоненти, вставлені в один ряд, виявляються з'єднаними.

Світлодіод (LED)

Світлодіод світиться, коли струм тече через нього в одному напрямку — від анода (+) до катода (−). Якщо переплутати — він просто не загориться. Розрізнити виводи легко:

  • Анод — довша ніжка, її підключають до керованого піна.
  • Катод — коротша ніжка з боку зрізаного корпусу, її підключають до «землі» (GND).

Резистор

Резистор — це штучна «перешкода» для струму. Він обмежує силу струму, щоб світлодіод не згорів. Опір вимірюється в омах (Ом). Для світлодіода нам підійде резистор на 220 Ом. Номінал зашифрований кольоровими смужками на корпусі.

💡 Чому 220 Ом?

Без резистора через світлодіод піде завеликий струм — і він швидко вийде з ладу. Резистор 220 Ом — безпечний «золотий стандарт» для більшості світлодіодів на 5 В.

🔌 Крок 1. Блимаємо одним світлодіодом

Підключимо світлодіод до піна 5: анод — через резистор 220 Ом до піна 5, катод — до GND.

Тепер код. Він змінює світлодіод між станами «увімкнено» (HIGH) і «вимкнено» (LOW) з різними паузами — і світлодіод блимає за власним ритмом.

blink.ino · Arduino C++
int ledPin = 5;            // пін світлодіода

void setup() {
  pinMode(ledPin, OUTPUT); // пін як «вихід»
}

void loop() {
  digitalWrite(ledPin, HIGH); // увімкнути
  delay(800);                 // чекати 800 мс
  digitalWrite(ledPin, LOW);  // вимкнути
  delay(400);
  digitalWrite(ledPin, HIGH);
  delay(150);
  digitalWrite(ledPin, LOW);
  delay(1000);
}

Покрокова практика

🧰 Реальний набір
  1. Від'єднай Arduino від USB.
  2. Встав світлодіод у макетну плату.
  3. З'єднай анод через резистор 220 Ом до піна 5.
  4. Катод з'єднай із піном GND.
  5. Підключи USB, завантаж код, натисни «Upload».
💻 Tinkercad
  1. Створи новий проєкт Circuits.
  2. Додай Arduino, макетну плату, світлодіод і резистор.
  3. Повтори з'єднання зі схеми вище.
  4. Встав код у редактор коду.
  5. Натисни «Start Simulation».

🚦 Крок 2. Збери «Світлофор»

Тепер головний проєкт. Додай три світлодіоди: червоний → пін 7, жовтий → пін 6, зелений → пін 5. Кожен — через свій резистор 220 Ом, як у попередній схемі.

Логіка: зелений світить 3 с → блимає 3 рази (тут і знадобиться цикл for) → жовтий 3 с → червоний 3 с → коротко спалахує жовтий → усе спочатку.

traffic_light.ino · Arduino C++
int redLed = 7;
int yellowLed = 6;
int greenLed = 5;

void setup() {
  pinMode(redLed, OUTPUT);
  pinMode(yellowLed, OUTPUT);
  pinMode(greenLed, OUTPUT);
}

void loop() {
  digitalWrite(greenLed, HIGH);
  delay(3000);
  digitalWrite(greenLed, LOW);

  // блимаємо зеленим 3 рази
  for (int i = 0; i < 3; i++) {
    delay(500);
    digitalWrite(greenLed, HIGH);
    delay(500);
    digitalWrite(greenLed, LOW);
  }

  digitalWrite(yellowLed, HIGH);
  delay(3000);
  digitalWrite(yellowLed, LOW);

  digitalWrite(redLed, HIGH);
  delay(3000);
  digitalWrite(yellowLed, HIGH);
  delay(1000);
  digitalWrite(redLed, LOW);
  digitalWrite(yellowLed, LOW);
}

🏆 Завдання-виклик (для портфоліо)

Використовуючи цикл, зроби «біжуче світло»: світлодіоди вмикаються по черзі від першого до третього, а потім гаснуть у зворотному напрямку. Збережи свій код або скріншот у портфоліо.
