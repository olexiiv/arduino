📖 Теорія коротко

Що таке Arduino UNO

Плата UNO — це маленький комп'ютер на основі мікроконтролера ATmega328. На ній уже є все потрібне: роз'єм USB, роз'єм живлення, кнопка перезавантаження та ряди контактів (їх називають пінами), до яких підключають деталі. Живлення йде через USB або блок на 7–12 В.

Піни поділяють на три групи:

  • Цифрові — мають лише два стани: є напруга (≈5 В, «одиниця») або немає (0 В, «нуль»). Їх 14, шість із них можуть плавно регулювати напругу (ШІМ).
  • Аналогові — уміють зчитувати напругу не лише «є/немає», а 1024 рівні від 0 до 5 В. Їх 6.
  • Живлення — контакти 5 В і 3.3 В («плюс») та кілька GND («земля», «мінус»).

💡 Головне про плату

Робоча напруга — 5 В, тактова частота — 16 МГц. Максимальний струм одного піна — 40 мА. Цього досить, щоб керувати світлодіодами, кнопками й багатьма датчиками.

Вбудований світлодіод

На платі вже є маленький світлодіод, позначений літерою «L». Він з'єднаний із піном 13. Саме з нього почнемо — жодних дротів не треба.

Як влаштована програма (скетч)

Будь-яка програма для Arduino складається з двох частин:

  • void setup() — виконується один раз під час запуску. Тут ми налаштовуємо піни.
  • void loop() — повторюється безкінечно, поки є живлення. Тут — основна робота.

Основні команди цього уроку:

  • pinMode(пін, OUTPUT) — налаштувати пін на «вихід» (щоб подавати напругу).
  • digitalWrite(пін, HIGH/LOW) — подати напругу (HIGH = 5 В) або прибрати (LOW = 0 В).
  • delay(мс) — пауза в мілісекундах (1000 мс = 1 с).

⚙️ Налаштування Arduino IDE

Щоб писати програми, потрібне середовище Arduino IDE (безкоштовне, з офіційного сайту arduino.cc). Перед завантаженням коду:

  1. Під'єднай плату USB-кабелем — на ній засвітиться зелений світлодіод ON.
  2. У меню Інструменти → Плата обери Arduino Uno.
  3. У меню Інструменти → Порт обери порт, до якого під'єднана плата.
  4. Напиши код і натисни Завантажити (Upload).

💡 У Tinkercad простіше

Онлайн-симулятор Tinkercad уже містить плату й редактор коду — нічого встановлювати не треба. Ідеально, щоб спробувати вдома або без набору.

💡 Крок 1. Блимаємо вбудованим світлодіодом

Наша перша програма вмикатиме й вимикатиме світлодіод із паузою в 1 секунду. Код у loop(), дійшовши до кінця, починається спочатку — і так безкінечно.

blink.ino · Arduino C++
int ledPin = 13;           // вбудований світлодіод «L»

void setup() {
  pinMode(ledPin, OUTPUT); // пін 13 — на вихід
}

void loop() {
  digitalWrite(ledPin, HIGH); // увімкнути (5 В)
  delay(1000);                // чекати 1 секунду
  digitalWrite(ledPin, LOW);  // вимкнути (0 В)
  delay(1000);                // ще секунда
}

🖥️ Крок 2. Виводимо повідомлення на комп'ютер

Мікроконтролер може «розповідати» комп'ютеру, що він робить, — через COM-порт. Це дуже корисно, щоб бачити, що відбувається всередині програми. Повідомлення з'являються у вікні Serial Monitor.

Спочатку в setup() запускаємо зв'язок командою Serial.begin(9600), а далі командою Serial.println(...) надсилаємо текст.

blink_serial.ino · Arduino C++
int ledPin = 13;

void setup() {
  pinMode(ledPin, OUTPUT);
  Serial.begin(9600); // запускаємо зв'язок із ПК
}

void loop() {
  digitalWrite(ledPin, HIGH);
  Serial.println("Світлодіод: увімкнено");
  delay(1000);
  digitalWrite(ledPin, LOW);
  Serial.println("Світлодіод: вимкнено");
  delay(1000);
}

💡 Де побачити текст

В Arduino IDE відкрий Інструменти → Serial Monitor (у Tinkercad — кнопка «Serial Monitor» під кодом). Переконайся, що швидкість — 9600.

🏆 Завдання-виклик (для портфоліо)

Сигнал SOS

Запрограмуй вбудований світлодіод передавати сигнал SOS азбукою Морзе: три коротких спалахи (· · ·), три довгих (− − −), три коротких (· · ·), а потім пауза — і спочатку. Підбери тривалості delay() для «крапки» та «тире». Збережи код або скріншот у портфоліо.
