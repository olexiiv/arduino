📖 Теорія коротко

Що таке RGB-світлодіод

RGB-світлодіод — це три світлодіоди в одному корпусі: червоний (Red), зелений (Green) і синій (Blue). Змішуючи ці три кольори в різній пропорції, можна отримати майже будь-який відтінок. Саме так працюють екрани телефонів і телевізорів.

У нього 4 ніжки: по одній на кожен колір і одна спільна. Найдовша ніжка — спільна; у нашому світлодіоді вона йде на «землю» (GND), тобто це спільний катод.

ШІМ — плавна яскравість

Цифровий пін уміє лише вмикати (5 В) і вимикати (0 В). Але якщо перемикати його дуже швидко — тисячі разів на секунду — око не помічає мерехтіння, і здається, що світлодіод горить не на повну. Це і є ШІМ (широтно-імпульсна модуляція, англ. PWM).

Чим довша «увімкнена» частина кожного циклу, тим яскравіше світло. В Arduino яскравість задають числом від 0 (вимкнено) до 255 (повна яскравість).

💡 Де є ШІМ

ШІМ працює не на всіх пінах, а лише на позначених символом ~ (тильда). На UNO це піни 3, 5, 6, 9, 10, 11. Саме тому наш RGB-світлодіод підключаємо до пінів 3, 5, 6.

🔌 Крок 1. Підключення і базові кольори

Підключи RGB-світлодіод через три резистори 220 Ом: червоний → пін 6, синій → пін 5, зелений → пін 3. Найдовшу (спільну) ніжку з'єднай із GND.

Керуємо кольорами так само, як звичайним світлодіодом — через digitalWrite. Але тепер, вмикаючи кілька кольорів одночасно, отримуємо нові: червоний + зелений = жовтий, а всі три разом = білий.

rgb_colors.ino · Arduino C++
int redPin = 6;
int bluePin = 5;
int greenPin = 3;

void setup() {
  pinMode(redPin, OUTPUT);
  pinMode(greenPin, OUTPUT);
  pinMode(bluePin, OUTPUT);
}

void loop() {
  // червоний
  digitalWrite(redPin, HIGH);
  digitalWrite(greenPin, LOW);
  digitalWrite(bluePin, LOW);
  delay(1000);

  // жовтий = червоний + зелений
  digitalWrite(greenPin, HIGH);
  delay(1000);

  // білий = усі три разом
  digitalWrite(bluePin, HIGH);
  delay(1000);
}

🌈 Крок 2. Плавні кольори через ШІМ

Тепер замість «увімк./вимк.» керуватимемо яскравістю кожного кольору окремо командою analogWrite(пін, значення), де значення — від 0 до 255. Так між основними кольорами з'являються тисячі відтінків.

У програмі нижче ми плавно переливаємо кольори за допомогою циклу for: червоний яскравішає, а зелений меркне — і так по колу.

rgb_fade.ino · Arduino C++
int redPin = 6;
int bluePin = 5;
int greenPin = 3;

void setup() {
  pinMode(redPin, OUTPUT);
  pinMode(greenPin, OUTPUT);
  pinMode(bluePin, OUTPUT);
}

void loop() {
  // червоний яскравішає, зелений меркне
  for (int i = 0; i <= 255; i++) {
    analogWrite(redPin, i);
    analogWrite(greenPin, 255 - i);
    delay(10);
  }
  // синій яскравішає, червоний меркне
  for (int i = 0; i <= 255; i++) {
    analogWrite(bluePin, i);
    analogWrite(redPin, 255 - i);
    delay(10);
  }
  // зелений яскравішає, синій меркне
  for (int i = 0; i <= 255; i++) {
    analogWrite(greenPin, i);
    analogWrite(bluePin, 255 - i);
    delay(10);
  }
}

💡 Скільки кольорів?

Кожен колір має 256 рівнів (0–255). Разом це 256 × 256 × 256 ≈ 16 мільйонів відтінків — стільки ж, скільки на екрані комп'ютера!

Палітра відтінків
