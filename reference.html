🧩 Компоненти

Світлодіод (LED)

Світиться, коли струм тече від анода (+) до катода (−). Анод — довша ніжка. Завжди підключай через резистор.

Резистор

Обмежує струм. Опір вимірюють в омах (Ом). Номінал зашифровано кольоровими смужками. Для світлодіода — 220 Ом.

Макетна плата

Дозволяє складати схеми без паяння. Краї — лінії живлення (+/−), центр — короткі ряди поперек плати.

Arduino UNO

Плата-мікроконтролер: «мозок» твоїх пристроїв. Має цифрові піни (D0–D13) і живлення (5 В, GND).

⌨️ Функції Arduino

pinMode(pin, OUTPUT)

Налаштовує пін як «вихід» (керуємо чимось) або «вхід» (зчитуємо). Викликають у setup().

digitalWrite(pin, HIGH)

Подає на пін напругу (HIGH) або вимикає її (LOW). Так вмикаємо/вимикаємо світлодіод.

delay(ms)

Пауза в програмі на задану кількість мілісекунд. delay(1000) — це 1 секунда.

for (...) { ... }

Цикл для повторення дій відому кількість разів. Має лічильник, умову та крок: for (int i = 0; i < 3; i++).

🛠️ Типові помилки

Найчастіше переплутані анод і катод. Переверни світлодіод. Перевір, чи резистор і всі дроти в правильних рядах макетної плати.

Перевір, чи обрано правильну плату й порт у меню «Інструменти». Спробуй інший USB-кабель або порт.

Найчастіше — забута крапка з комою ; в кінці рядка або незакрита дужка { }. Уважно перечитай рядок, на який вказує помилка.

📖 Глосарій

Анод

Вивід «+» світлодіода (довша ніжка).

Катод

Вивід «−» світлодіода, іде на GND.

GND

«Земля» — нульова точка схеми.

Струм

Рух заряджених частинок у колі.

Напруга

«Тиск», що штовхає струм. Вимір — вольт (В).

Пін

Контакт на платі, до якого підключають дроти.
